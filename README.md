# Online_Bookstore
数据库原理实验——基于python和SQLite的网上书店，采用B/S架构

## 第一步：确认文件结构
首先，找到你之前整理好的项目文件夹（例如 `我的云书店`），确保里面的文件结构完整：

```text
我的云书店/
├── app.exe                  <-- 你的核心程序
├── ngrok.exe                <-- 内网穿透工具（建议把ngrok也放这里，方便操作）
├── instance/                <-- 存放数据库的文件夹
│     └── cloud_bookstore_real.db
└── images/                  <-- 图片文件夹
      ├── Python编程从入门到实践.jpg
      └── ...
```

第二步：本地功能测试（基础）
这一步用于验证程序本身是否正常，数据库读取和图片显示是否无误。
启动服务器：
双击文件夹中的 app.exe。
现象：会弹出一个黑色的命令行窗口。
检查：窗口内显示 Running on http://0.0.0.0:5000 或类似字样。
注意：千万不要关闭这个黑框！ 它是你的服务器心脏。
访问网页：
打开浏览器（Chrome/Edge）。
在地址栏输入：http://127.0.0.1:5000
验证点：
1.首页能否正常打开？
2.图书封面图片是否显示正常？（不再是裂开的图标）
3.尝试点击“登录”，输入 admin / 123456（或其他已注册账号）能否登录？

第三步：公网/手机访问测试（进阶）
这一步用于模拟真实互联网环境，让你的手机或朋友也能访问。
保持 app.exe 正在运行（即上一步的黑框不要关）。
启动穿透工具：
双击 ngrok.exe。
在弹出的新黑框中输入以下命令并回车：

ngrok http 5000
获取链接：
查看屏幕上的 Forwarding 一行，复制类似这样的链接：
https://xxxx-xxxx-xxxx.ngrok-free.app
手机测试：
关闭电脑的 WiFi（或者手机使用 4G/5G 流量），确保手机和电脑不在同一个网络下。
在手机浏览器中输入刚才复制的 https://... 链接。

第四步：结束测试
测试完成后，如何正确关闭程序：
关闭网站：直接点击 app.exe 的黑框右上角的 X 关闭即可。
关闭穿透：直接点击 ngrok.exe 的黑框右上角的 X 关闭即可。
